Sass是CSS的預處理器

瀏覽器並不認識Sass!

安裝方式

  1. 在命令列執行: gem install sass
  2. 使用codekit, compass.app等桌機軟體
  3. 執行: sass input.sass output.css
  4. 在瀏覽器上使用"output.css"

參考官方網站: http://sass-lang.com/install

有兩種格式

.sass 與 .scss

變數 Variables

  • 使用$符號作為變數，例如: $base_color: #C$0000
  • 型別有文字, 數字, 布林值, 單位, 顏色, 多組文字與空值
  • 可以同時應用在"鍵"與"值"上
  • 需要注意命名的方式
  • 透過partial可在多檔案中使用
  • 配合mixin發揮強大作用

數字計算 Operators

數字計算的函數

Sass內建好用的函數，提高數學計算的可能性

  • round($number) => 四捨五入
  • ceil($number) => 無條件進位
  • floor($number) => 無條件去小數位
  • abs($number) => 絕對值
  • percentage($number) => 數值百分位化
  • min($collection) => 一堆值中最小的
  • max($collection) => 一堆值中最大的
